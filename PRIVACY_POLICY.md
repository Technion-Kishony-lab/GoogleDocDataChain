# Privacy Policy

**Data Chain** is a Google Docs add-on developed by Technion Kishony Lab.

## Data Collection
This add-on does **not** collect, store, or transmit any personal data to external servers.

## Data Access
The add-on accesses only:
- Google Docs documents you explicitly use with the add-on
- Google Sheets spreadsheets you explicitly select

All processing occurs within Google's infrastructure under your account.

## Third Parties
No data is shared with third parties.
