Data Chain is an add-on that lets you insert linked values from a Google Sheet into your Google Doc.

How to Use

  • From the "Dataâ€‘Chain" menu, select Insert value.
  • Choose a spreadsheet, tab, and field.
  • Click Insert to place the value at the cursor.
  • Click Update document to refresh all inserted values.

Spreadsheet Structure

The add-on expects field-value pairs in the second and third columns. Your spreadsheet should follow this format:

| Description    | Field    | Value |
|----------------|----------|-------|
| Number of cats | num_cats | 123   |
| Number of dogs | num_dogs | 456   |

New Spreadsheet ID

To insert from a new spreadsheet, choose New in the Insert chained value sidebar, and insert the speadsheet ID.

To find the speadsheet ID:

  • Open your Google Sheet.
  • In the URL, find the ID between /d/ and /edit.
  • Paste that ID into the Spreadsheet ID box in the sidebar.
